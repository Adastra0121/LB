#### 题目描述

如果一个正整数的十进制表示的每位数码都不同，则称它为独特数。

将所有独特数从小到大排列，给定 n，请求出第 n 小的独特数（0 不算独特数）。

#### 输入

单个整数：表示 n。

#### 输出

单个整数：表示第 n 个独特数。

#### 数据范围

-   对于 50% 的数据，1≤n≤1000
-   对于 100% 的数据，1≤n≤100,000

___

#### 输入样例1复制

```
1
```

#### 输出样例1复制

```
1
```

___

#### 输入样例2复制

```
11
```

#### 输出样例2复制

```
12
```

#### 样例解释2

`1`到`12`之间只有`11`不是独特数
