#### 题目描述

最近老师要装饰酷思的照片墙，小酷请同学帮忙拍了三张照片，但他只能选择一张粘贴在照片墙上。小酷很纠结，所以邀请同学们帮忙投票选择哪张照片最受欢迎。

下面请你写一个程序，有 n 位同学参与投票，依次输入同学们选择的照片编号，并输出结果。  

如果有相同的票数，输出最靠前的结果。  

#### 输入

第一行一个整数 n(n≤1000)；  
第二行 n 个 1～3 之间的整数用空格隔开，表示投票的照片编号。  

#### 输出

一行 2 个整数，分别是最受欢迎的照片编号及得票数。

___

#### 输入样例复制

```
6
1 1 2 3 2 2
```

#### 输出样例复制

```
2 3
```
