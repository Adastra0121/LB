#### 题目描述

判断一个正整数 n 是否能被一个“幸运数”整除。幸运数是指一个只包含 4 或 7 的正整数，如 7，47，477 等都是幸运数 17，42 则不是幸运数。  

#### 输入

一行一个正整数 n(1≤n≤1000)。

#### 输出

一行一个字符串，

-   如果能被幸运数整除则输出`YES`；
-   否则输出`NO`。

___

#### 输入样例复制

```
47
```

#### 输出样例复制

```
YES
```

#### 温馨提示 复制

```
4,7,44,47,74,77,444,447,474,477,744,747,774,777
```
