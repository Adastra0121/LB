#### 题目描述

酷星发生了洪灾。小酷发完了赈灾物资，接下来还得去统计受灾的影响程度，真是辛苦啊。假定酷星为 N×N 个区域，每个区域的水位(深度)已知，我们划定一块 2×2 的区域为受灾最严重区域。

#### 输入

对于每组数据存在 N（2≤N≤10）表示区域边宽度。

接下来有 N×N 个整数，表示各个区域的水位(深度)。

#### 输出

对于每组数据，我们输出受灾最严重区域的平均水位(深度)，保证数据范围都在 2^{31} 内，并且保留两位小数。

___

#### 输入样例复制

```
3
1 2 3
2 3 4
3 4 5
```

#### 输出样例复制

```
4.00
```
