#### 题目描述

给出一个正整数 n，定义：

-   $sum_1=1^2+2^2+...+n^2$
-   $sum_2=1^3+2^3+...+n^3$

请你输出 $sum_1$ 和 $sum_2$ 的值。  

输出的格式要求为，每个整数宽度长 10，不足 10 位需要补前导 0，请用`printf`完成这一要求。  

#### 输入

一个正整数 n(1 ≤ n ≤ 200) 。

#### 输出

按指定格式输出两个十进制整数，按空格分隔。

___

#### 输入样例复制

```
3
```

#### 输出样例复制

```
0000000014 0000000036
```
