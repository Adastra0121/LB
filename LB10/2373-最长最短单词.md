#### 题目描述

输入 1 行句子（不多于 200 个单词，每个单词长度不超过 100），只包含字母、空格。单词由至少一个连续的字母构成，空格是单词间的间隔。

试输出第 1 个最长的单词和第 1 个最短单词。

#### 输入

一行句子。

#### 输出

两行输出：

-   第 1 行，第一个最长的单词。
-   第 2 行，第一个最短的单词。

#### 提示

如果所有单词长度相同，那么第一个单词既是最长单词也是最短单词。

___

#### 输入样例复制

```
I am studying Programming language C in Peking University
```

#### 输出样例复制

```
Programming
I
```
